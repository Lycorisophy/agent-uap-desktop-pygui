## 核心模块详解

### A. ReAct Agent (`src/uap/react/react_agent.py`)

**执行流程**：
```
用户消息 → LLM思考(Thought) → 选择技能(Action) → 执行 → 观察(Observation) → 循环
```

**关键类**：
- `ReactAgent` - 主引擎
- `SkillSession` - 会话状态
- `ReactStep` - 单步执行记录

**调用方式**：
```python
from uap.react import ReactAgent, DstManager

agent = ReactAgent(
    llm_client=llm_client,
    skills_registry=skills,
    dst_manager=dst_manager,
    max_iterations=15
)
result = agent.run(user_message, context)
```

### B. DST管理器 (`src/uap/react/dst_manager.py`)

**7个建模阶段**：
```
INITIAL → INTENT_DETECTION → VARIABLE_COLLECTION → 
RELATION_DISCOVERY → CONSTRAINT_DEFINITION → 
MODEL_VALIDATION → COMPLETED
```

**用途**：追踪智能体建模进度，实时更新UI

### C. 技能系统

**基类**：`AtomicSkill`（来自`uap/skill/atomic_skills.py`）

**文件访问技能**：
```python
from uap.react.file_access_skill import create_file_access_skill

file_skill = create_file_access_skill(
    project_folder="/path/to/project"
)
result = file_skill.execute(action="read", path="intro/README.md")
```

### D. 项目服务 (`src/uap/service/project_service.py`)

**关键方法**：
```python
react_modeling(project_id, user_message, card_manager, web_search_func)
# 返回：{ok, message, model, session_id, steps, dst_state, pending_card}
```

## 配置说明

### LLM配置
**位置**：`src/uap/config.py`

```python
llm:
  provider: "ollama"
  model: "llama3.2"          # 默认模型
  base_url: "http://localhost:11434"
  embedding_model: "nomic-embed-text"
```

### 存储路径
```python
storage:
  projects_root: "~/.uap/projects"  # 项目根目录
```

每个项目创建时生成子目录：
```
~/.uap/projects/{project_id}/
├── intro/          # 项目介绍
├── skills/         # 技能配置
├── logs/           # 运行日志
├── models/         # 模型文件
├── tasks/          # 定时任务
└── data/           # 数据文件
```
